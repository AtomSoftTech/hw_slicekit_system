
STAR, SQUARE SLOT Pinout
------------------------

Star   = XCoreTile #0
Square = XCoreTile #1

+----+-----------------+------------------+
|pin | Side B (Top)    | Side A (Bottom)  |
+====+==========+======+==========+=======+
|    | Function | Port | Function | Port  |
+----+----------+------+----------+-------+
| 1  |          |      |          |       |
+----+----------+------+----------+-------+
| 2  |          |      | 5V       |       |
+----+----------+------+----------+-------+
| 3  | GND      |      |          |       |
+----+----------+------+----------+-------+
| 4  |          |      |          |       |
+----+----------+------+----------+-------+
| 5  | 3V3      |      |          |       |
+----+----------+------+----------+-------+
| 6  | RXD0     | 8A0  | RXD6	  | 8A6	  |
+----+----------+------+----------+-------+
| 7  | RXD1     | 8A1  | RXD7  	  | 8A7	  |
+----+----------+------+----------+-------+
| 8  | GND      |      |	  |  	  |
+----+----------+------+----------+-------+
| 9  | RXD2     | 8A2  | RXD4	  | 8A4	  |
+----+----------+------+----------+-------+
| 10 |	        | 1C   | GND	  |	  |
+----+----------+------+----------+-------+
| 11 | RXD3     | 8A3  | RXD5	  | 8A5	  |
+----+----------+------+----------+-------+
|    |	        |      |	  | 	  |
+----+----------+------+----------+-------+	
| 12 | TXD0     | 8B0  | TXD6	  | 8B6	  |
+----+----------+------+----------+-------+
| 13 | TXD1     | 8B1  | TXD7	  | 8B7	  |
+----+----------+------+----------+-------+
| 14 | CLK      |      | GND	  |       |
+----+----------+------+----------+-------+		
| 15 |	        |      | CLK_OUT  | 1F    |
+----+----------+------+----------+-------+
| 16 | GND      |      | RST_N    |	  |
+----+----------+------+----------+-------+			
| 17 | TXD2     | 8B2  | TXD4	  | 8B5	  |
+----+----------+------+----------+-------+
| 18 | TXD3     | 8B3  | TXD5	  | 8B6	  |
+----+----------+------+----------+-------+

|pin | Side B (Top)    | Side A (Bottom)  |
